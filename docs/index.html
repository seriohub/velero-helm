Helm Chart Repository

Welcome to the seriohub-velero Helm chart repository.

To use this repository, add it to Helm using the following command:

helm repo add seriohub-velero https://seriohub.github.io/velero-helm/

Then, you can search and install charts:

helm search repo seriohub-velero

For installation instructions, please visit the following page:

Helm Installation Guide
